⚙️ Conectar con Google Sheets
Solo necesitas hacer esto una sola vez. Después el dashboard se actualiza automáticamente cada vez que abras el archivo.
En tu Google Sheets ve a Extensiones → Apps Script
Borra todo el código existente y pega el contenido del archivo apps_script.js que viene junto con este dashboard
Haz clic en Guardar 💾 y luego en Implementar → Nueva implementación
Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
Clic en Implementar, autoriza los permisos y copia la URL que aparece
Pega esa URL aquí abajo y haz clic en Conectar
🔒 La URL solo se guarda en este navegador. Nadie más puede ver tus datos a menos que tú les compartas el archivo HTML.
